## Event microservice (API + Lambdas)

Create event payload structure:
``` js
{
  "game":"game name",
  "hostName":"host name",
  "eventDate":"04-22-2021 12PM",
  "price":"$10",
  "address":"222",
  "registration":"registration steps",
  "description":"this is a description",
  "stream":"this is a stream url"
}
```
